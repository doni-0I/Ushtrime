## Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, Edge)
- A text editor or IDE (VS Code recommended)

### Running the Project

Simply open any `.html` file in your web browser:

1. Navigate to the project directory
2. Right-click on any `.html` file
3. Select "Open with" and choose your preferred browser
4. Or drag and drop the file into your browser

## Usage

Each HTML file (`U1.html`, etc.) contains practice exercises. Review the code to understand the concepts and modify them as needed for learning purposes.
